# Super Ask

[中文](#中文) | [English](#english)

---

## English

Super Ask enables AI to report progress and collect user feedback during task execution, supporting multi-round feedback loops within a single request.

### Components

| Component | Description | Status |
|-----------|-------------|--------|
| [VSCode Extension](./vscode/) | Language Model Tool for VSCode + Copilot Agent Mode | Available |

### Quick Start

```bash
cd vscode
bash build.sh
code --install-extension super-ask-1.0.0.vsix
```

See the [VSCode Extension README](./vscode/README.md) for detailed usage instructions.

Super Ask 让 AI 在任务执行过程中向用户汇报进度并获取反馈，支持在同一个 request 中实现多轮反馈循环。

### 组件

| 组件 | 说明 | 状态 |
|------|------|------|
| [VSCode 插件](./vscode/) | VSCode + Copilot Agent Mode 的 Language Model Tool | 可用 |

### 快速开始

```bash
cd vscode
bash build.sh
code --install-extension super-ask-1.0.0.vsix
```

详细使用说明见 [VSCode 插件 README](./vscode/README.md)。
