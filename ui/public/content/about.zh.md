## 产品简介

super-ask 是一个面向 AI 编程 Agent 的多轮人机交互中间件，让 Agent 在执行过程中可以随时暂停、汇报进展、提出问题，并在收到用户反馈后继续工作。

它的产品定位是为不同 IDE、不同 Agent 提供统一的反馈协作层，通过同一套协议、CLI 与 Web UI，把任务中的汇报、提问、等待和继续串成可追踪的闭环。

## 特色功能

- **阻塞式交互**：Agent 调用后会暂停等待，用户回复后自动继续执行。
- **统一 Web UI 仪表盘**：集中查看多个 Agent 会话、来源与状态。
- **多平台部署**：支持为 Cursor、Copilot、Codex、OpenCode、Qwen CLI 等环境一键部署规则。
- **高效回复能力**：支持预定义消息、回复队列和文件附件。
- **高效消息管理**：支持一键消息引用、复制、收藏。
- **中英文双语界面**：便于不同团队成员协作与使用。
