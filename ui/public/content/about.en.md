## Product Introduction

super-ask is multi-turn human-in-the-loop middleware for AI coding agents. It lets agents pause at any time during execution, report progress, ask questions, and continue working after receiving user feedback.

It is positioned as a unified feedback and collaboration layer for different IDEs and agents. With one protocol, one CLI, and one Web UI, it turns reporting, asking, waiting, and continuing into a traceable closed loop within a task.

## Features

- **Blocking interaction**: when the agent invokes super-ask, execution pauses and waits; after the user replies, it continues automatically.
- **Unified Web UI dashboard**: view multiple agent sessions, their sources, and status in one place.
- **Multi-platform deployment**: one-click deployment of rules for environments such as Cursor, Copilot, Codex, OpenCode, and Qwen CLI.
- **Efficient replies**: predefined messages, a reply queue, and file attachments.
- **Efficient message handling**: one-click quoting, copying, and favoriting of messages.
- **Chinese and English UI**: easier for different team members to collaborate and use.
